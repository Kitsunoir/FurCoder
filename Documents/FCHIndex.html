The FurCoderCXX is a graphical application for the creation/decyphering of The Code of the Furries, by Ross Smith: originated 23 June, 1996. The Fur Code is up to version: 1.3 (7 Feburary, 1998) The "Fur Code" is based upon the "Geek Code" (Robert Hayden: 1993) that goes back to the Usenet days. It is a method of conveying details concerning some personal matters and describing the level of one's participation within Furry fandom with as concisely as possible. While such considerations as limiting the amount of text involved in on-line communications is not so important these days, these codes have remained quite popular as a "sig" item for on-line forums and web pages. There are a variety of these codes out there, serving a number of different on-line communities. As with the "Geek Code", the "Fur Code" has the advantages of conciseness, being incomprehensible to the "mundanes" (i.e. outsiders) and it's a really kewel addition to any on-line forum sig.

FurCoderCXX allows the user to easily decipher a Fur Code entered as a text file, via copy 'n' paste, or manually. The results are displayed in the main window, and may be saved to disk. Encode mode allows you to create a Fur Code easily by menu selection with a minimum of manual entries required. Again, all intermediate and final results are displayed, and the user has the options to save to disk, or copy from the display. Codes saved to disk may also be loaded for copy 'n' paste operations.

FurCoderCXX was written with the FOX GUI toolkit, and XEMACS on a Slackware 10.1 i686 System.

Main Window: Overview

 

 

Main Menubar:

File:

I)New Project: Reset the app to its initial state to start a new project. Will prompt if the current project hasn't been saved.
II)Open: Open a file; read into code display window
III)Save as: Save the current project to disk
V)Exit: Shut down the app; will prompt if current project hasn't been saved
Note: The main menubar is dock-able/free-floating.

Edit:
I)Cut: Cut selected text to the system clipboard
II)Copy: Copy selected text to the system clipboard
III)Paste: Paste text from the clipboard into a text widget
IV)Delete: Remove the selected text

Help:

I)Contents: Display the on-line help
II)About: Show the "About" box
III)Dolphins: Show a special message

Toolbar:

I) New Project
II) File Open
III) Save as
IV) Cut
V) Copy
VI) Paste
VII) Delete
VIII) Toggle Main Mode
IX) On-line Help
X) "About" box

Both the main menu and tool bar are fully relocatable and free floating.

Main Window Panes:

I) Option menus used for code creation
II) Optional code modifier command button group (encode)/Code display area (decode)
Note: The vertical divider between the panes is draggable in order to allow resizing of these panes. This can help improve readability of either pane.

Command Button Group:
I)Next: In encode mode: moves from the current menu group to the next (six total); finalizes the current selections.
II)OK: Accept the current menu selections (encode). Start decoding a Fur Code (decode mode)
III)Cancel: Reject current selection; select again.
IV)Exit: Shut down the app, prompt if there's an unsaved project

Statusbar:

I)Verbose tooltip display area
II)Show mode/menu selection status
III)Command button: show a special message
